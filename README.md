BassNote
========

Visual help for Bass guitar notes

Made with <3 from MontrÃ©al

###Songs format :
- Notes : ```g10``` Cord "G", 10th fret.
- Silence : ```--``` will create an empty space.
- Vertical line : ```|``` will create a line in all the strings. 

eg. : 
```["a7", "d6", "d7", "--", "d7", "|"]```
will create : 
```
G|-----------|
D|---6-7---7-|
A|-7---------|
E|-----------|
```
